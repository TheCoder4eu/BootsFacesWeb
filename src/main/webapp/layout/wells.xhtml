Wells

Use the well as a simple effect on an element to give it an inset effect.

Well Header

Well, that's the content of the <b:well />. Markup:

value vs. content

You can use the attribute value to put a simple text into the alert body. Alternatively, you can put the content between the opening <b:well /> and the closing </b:well /> tag. This allows for arbitrary complex content.

Reference section

Skinning

A Well is simply a div bearing the CSS class well. If a size is given, a second CSS class is added:

  • well-lg if size="lg"
  • well-sm if size="sm"
