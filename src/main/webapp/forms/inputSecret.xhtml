Password fields: <b:inputSecret />

<b:inputSecret> is basically a <b:inputText> which doesn't show the text.

So simply refer to the documentation of <b:inputText>.

Since BootsFaces 1.1.2, the value attribute is not rendered. The disadvantage is that the user has to key in their password a second time if the validation failed at first try. But from a security point of view, it's better that way. If you're using <b:inputSecret>, please update to BootsFaces 1.1.2 (or higher) if you're using an older version.
